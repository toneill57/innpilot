## 3. Deployment Manual

### Proceso Completo

1. **SSH al Servidor**:
   ```bash
   ssh root@muva.chat
   # O con alias: ssh innpilot-vps
   ```

2. **Navegar al Directorio**:
   ```bash
   cd /var/www/innpilot
   ```

3. **Pull Latest Changes**:
   ```bash
   git pull origin dev
   # O para forzar (sobrescribe cambios locales):
   # git fetch origin dev
   # git reset --hard origin/dev
   ```

4. **Instalar Dependencias**:
   ```bash
   npm ci
   # Nota: usa 'ci' (no 'install') para deps exactas de package-lock.json
   ```

5. **Build Application**:
   ```bash
   npm run build
   ```

6. **Restart PM2**:
   ```bash
   pm2 reload innpilot --update-env
   # O para restart completo (downtime breve):
   # pm2 restart innpilot
   ```

7. **Verificar Deploy**:
   ```bash
   # Health check
   curl -s http://localhost:3000/api/health | jq

   # PM2 status
   pm2 status innpilot

   # Nginx status
   sudo systemctl status nginx
   ```



